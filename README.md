# 🤖 AI Chatbot using Python & Tkinter

## 📌 Features
- User login interface
- Chatbot UI
- Real-time AI responses
- Simple GUI using Tkinter

## 🛠️ Technologies Used
- Python
- Tkinter
- API Integration

## 🚀 How to Run
1. Install Python
2. Install requests module
3. Run the file:
   python chatbot.py
